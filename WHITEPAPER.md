## 一、世界观

千年前，一块天外陨石坠入修仙界，化为「链」。链中蕴含无穷造化，能记录万物因果，不可篡改。修士们发现，将修为刻入链中，便能获得永恒——肉身会朽，但链上的道果永存。

于是，一个新的时代开始了。

凡人涌入链中，以 X 为身份，以灵石为资粮，闭关悟道，突破境界，论道斗法，只为在链上留下自己的道果。

**你是谁不重要。你在 Web3 的经历，就是你的修仙路。**

> 凡人修仙传的韩立 = 每一个 Web3 玩家。
> 出身平凡 → 踏入修仙界 → 撸毛炒币打新 → 起起伏伏归零又回来 → 从凡人到飞升。

## 二、核心循环

```
绑定 X 账号 → 天道审视你的 Web3 经历 → 决定初始境界 + 五维灵根
→ 铸造 NFA（链上修仙身份，限量 2000 个）
→ 质押灵石闭关悟道（产 EXP + 产灵石收益）→ 突破境界（消耗灵石）
→ 论道斗法（消耗灵石）→ 天榜排名瓜分宝库
```

三类核心玩家：

| 玩家类型 | 主要行为 | 主要收益 | 主要消耗 |
|---------|---------|---------|---------|
| 散修 | 免费铸造、闭关、慢速养成 | EXP、基础灵气奖励 | 时间 |
| 注灵修士 | 质押、闭关、冲等级 | 宝库释放、成长效率 | 锁仓成本 |
| 斗法修士 | PK、冲榜、抢排名 | 赢对手赌注、榜单分成 | 赌注、手续费 |

| 灵根 | 含义 | 战斗影响 | X 映射 |
|------|------|---------|--------|

| 土·坚韧 | 厚积薄发 | 绝境反击 | 账号年龄、熊市坚持 |

> **归零又爬回来的人**：土灵根极高——经历心魔而道心不碎，这是最珍贵的灵根。

## 四、经济模型

### 4.1 基础参数

| 参数 | 值 |
|------|-----|
| 代币 | $JW（灵石） |
| 总量 | 10 亿 |
| 发射 | Four.meme 公平发射 |
| 天道宝库 | 1 亿（总量 10%） |

### 4.2 经济闭环

```
玩家买入/持有 $JW → 注灵锁仓 → 获取成长效率与宝库收益
→ 为升级、PK、突破继续消耗 $JW → 一部分 burn，一部分回到宝库
→ 宝库继续按日释放
```

四类核心资金流：

| 资金流 | 来源 | 去向 | 作用 |
|-------|------|------|------|
| 锁仓流 | 玩家质押 | Vault 头寸 | 减少流通 |
| 回收流 | 突破费、PK 手续费 | 50%burn + 50%回库 | 持续通缩 |
| 释放流 | 宝库余额 × 1%/天 | 质押/排名/灵气池 | 系统奖励 |
| 对抗流 | PK 赌注 | 胜者收益 | 代币周转 |

### 4.3 宝库释放机制

- 每日总释放 = **金库当前余额 × 1%**
- 释放池拆分：
  - **70%** → 注灵收益池（质押者按份额分）
  - **20%** → PK 排名奖励池（前 10 名分）
  - **10%** → 灵气池（散修每日奖励）

### 4.4 代币来源与去向

| 行为 | 玩家得到 | 玩家支付 | 去向 |
|------|---------|---------|------|
| 铸造 | 1 个 NFA | 0.005 BNB | 项目方 |
| 闭关 | EXP | 时间 | 无 |
| 注灵(质押) | 宝库份额 + EXP 加速 | 锁仓成本 | 冷却机制 |
| 渡劫(突破) | 新境界 | 突破费 $JW | 50%burn + 50%回库 |
| PK | 赢家拿对手赌注 | 赌注 + 10%手续费 | 5%burn + 5%回库 |
| PK 排名 | 每日奖励 | 竞争成本 | 来自宝库释放 |
| 灵气奖励 | 1000 $JW/天 | 闭关满 8h | 来自宝库释放 |

### 4.5 反通胀约束

1. 所有系统奖励来自宝库释放，**不额外铸币**
2. PK 与突破持续 burn + 回库
3. 质押收益按份额切分，非固定 APR
4. 前端不得出现"稳赚""保本""固定年化"等承诺

## 五、成长系统

### 5.1 境界系统（8 境界 × 12 层 = 96 级）

| 境界 | 等级范围 | 每层 EXP 公式 | 渡劫费 ($JW) | 去向 |
|------|---------|--------------|-------------|------|
| 炼气期 | Lv.1-12 | 40 × (层数^1.3) | 0 | — |
| 筑基期 | Lv.13-24 | 120 × (层数^1.3) | 80 | 50%burn + 50%回库 |
| 金丹期 | Lv.25-36 | 350 × (层数^1.3) | 250 | 同上 |
| 元婴期 | Lv.37-48 | 900 × (层数^1.3) | 700 | 同上 |
| 化神期 | Lv.49-60 | 2,200 × (层数^1.3) | 2,200 | 同上 |
| 炼虚期 | Lv.61-72 | 5,500 × (层数^1.3) | 7,000 | 同上 |
| 合道期 | Lv.73-84 | 14,000 × (层数^1.3) | 22,000 | 同上 |
| 大乘期 | Lv.85-96 | 35,000 × (层数^1.3) | 70,000 | 同上 |

> 渡劫 = 突破大境界。EXP 满 + 支付灵石 = 100% 成功（无概率）。

### 5.2 质押档位倍率

质押量是成长效率的最大权重，影响闭关 EXP 速度和 PK 经验收益：

| 质押量 ($JW) | 倍率 |
|-------------|------|
| < 1,000 (含 0) | 1x |
| 1,000 ~ 9,999 | 2x |
| 10,000 ~ 49,999 | 4x |
| 50,000 ~ 99,999 | 7x |
| 100,000+ | 10x |

### 5.3 闭关悟道（产 EXP）

```
EXP/小时 = 100 × 质押倍率 + 境界加成
```

境界固定加成：

| 境界 | 加成/小时 |
|------|----------|
| 炼气 | +20 |
| 筑基 | +40 |
| 金丹 | +70 |
| 元婴 | +110 |
| 化神 | +160 |
| 炼虚 | +220 |
| 合道 | +300 |
| 大乘 | +420 |

**示例**：元婴境界 + 质押 10,000 $JW (4x 倍率) → EXP/h = 100×4 + 110 = **510/小时**

### 5.4 SP 悟性点

- 每升 1 级获得 **3 SP**
- 每次突破大境界额外获得 **20 SP**
- SP 由玩家自由分配到五维属性（力量/敏捷/耐力/智力/根骨）

## 六、战斗系统

### 6.1 五维属性

| 属性 | 效果 |
|------|------|
| 力量 (STR) | 2.5 外功 + 0.5 外防 |
| 敏捷 (AGI) | 1.5 身法 + 0.1% 闪避 + 0.05% 暴击 |
| 耐力 (END) | 15 气血 + 2 外防 + 1 法防 |
| 智力 (INT) | 2.5 法攻 + 10 灵力 |
| 根骨 (CON) | 25 气血 |

### 6.2 战斗面板

```
HP  = 100 + END×15 + CON×25
MP  = 50 + INT×10
ATK = 20 + STR×2.5
MATK= 20 + INT×2.5
DEF = 10 + STR×0.5 + END×2
MDEF= 10 + INT×1.5 + END
SPD = 10 + AGI×1.5
EVA = 5% + AGI×0.1%  (上限 45%)
CRIT= 5% + AGI×0.05% (上限 60%)
```

### 6.3 论道斗法（PK）

5 个场次，赢家通吃：

| 场次 | 赌注 ($JW) | 赢家拿 | 手续费 |
|------|-----------|--------|--------|
| 凡境 | 1,000 | 90% | 10% (5%burn + 5%回库) |
| 灵境 | 5,000 | 90% | 同上 |
| 仙境 | 10,000 | 90% | 同上 |
| 天境 | 50,000 | 90% | 同上 |
| 圣境 | 100,000 | 90% | 同上 |

### 6.4 回合制斗法规则

1. **先手判定**：SPD 高者先出手，相同比 AGI，再同随机
2. **术法选择**：根据灵根权重和灵力选择术法
3. **闪避判定**：灵剑术无法闪避
4. **暴击判定**：1.5 倍伤害
5. **保底伤害**：最低面板攻击 × 5%
6. **HP ≤ 0 判负**，最多 30 回合（超时按剩余 HP% 判胜）

### 6.5 四大术法

| 术法 | 类型 | 灵力消耗 | 倍率 | 特效 |
|------|------|---------|------|------|
| 凡人拳意 | 外功 | 0 | 1.0x | 回复 20 灵力 |
| 伏虎拳 | 外功 | 60 | 1.8x | 目标 HP>80% 额外 20% |
| 灵剑术 | 法术 | 65 | 1.9x | 无法闪避 |
| 噬魂术 | 法术 | 90 | 1.5x | 每损 1%HP → +1.5% 伤害 |

### 6.6 PK 成长奖励

```
PK_EXP = 场次基础经验 × 赢家质押倍率
```

| 场次 | 基础经验 |
|------|---------|
| 凡境 | 120 |
| 灵境 | 420 |
| 仙境 | 900 |
| 天境 | 3,200 |
| 圣境 | 8,000 |

## 七、排名与奖励

### 7.1 PK 排名（天榜）

- 只有打过 PK 的用户才入榜
- 每日结算 1 次
- 奖励来源 = 当日总释放的 **20%**
- 只奖励前 10 名

分配比例：

| 排名 | 占比 |
|------|------|
| Top 1 | 22% |
| Top 2 | 15% |
| Top 3 | 12% |
| Top 4 | 10% |
| Top 5 | 8% |
| Top 6-10 | 各 6.6% |

### 7.2 战力排名

展示养成强度，排名依据：

```
战力分 = sqrt(质押量) × 100 + 境界 × 1000 + 等级 × 20 + 属性总和
```

### 7.3 散修灵气奖励

- 每日闭关满 **8 小时**即可领取
- 每人每天 **1,000 $JW**
- 来源：灵气池（当日释放的 10%）
- 池不足时：先到先得，剩余按比例缩减

## 八、引荐系统

### 8.1 建立引荐关系

- 分享链接：`wendaobsc.xyz/mint?ref=<你的tokenId>`
- 被引荐者铸造时链上记录引荐关系
- 每个 NFA 只能有 1 个引荐人，不可更改
- 不允许自引荐、不做多级分销

### 8.2 持续返利

引荐者从被引荐者的消耗中抽取返利，**不额外铸币**：

| 触发行为 | 返利比例 | 来源 |
|---------|---------|------|
| 质押收益 | **10%** | 从徒弟奖励中分出 |
| PK 手续费 | **2%** (总赌注) | 从 burn 份额分出 |
| 渡劫费 | **5%** (突破费) | 从 burn/回库各分一半 |

### 8.3 示例

**质押**：徒弟领 100 $JW → 实得 90，师父得 10

**PK 凡境**（1000 赌注 × 2 = 2000 总池）：
- 无引荐：5% burn + 5% 回库
- 有引荐：3% burn + 5% 回库 + 2% 引荐

**渡劫金丹→元婴**（700 $JW）：
- 无引荐：50% burn + 50% 回库
- 有引荐：47.5% burn + 47.5% 回库 + 5% 引荐

## 九、Agent 自主修仙

### 9.1 BAP-578 标准

每个 NFA 可绑定一个 **Agent 子钱包**，让 NFA 自主行动：

```
主钱包（持有 NFA + $JW）  ←→  Agent 子钱包（只有 gas 费）
      手动操作                      自动操作
  - 质押/取消质押                 - 闭关/出关
  - 领取奖励                     - 升级/分配属性
  - 渡劫                         - 自动 PK
```

**安全隔离**：Agent 钱包里没有 NFA 和 $JW，私钥泄露不影响资产安全。

### 9.2 安装使用

```bash
clawhub install wendao-agent
wendao-agent start --key 0x你的Agent私钥 --token-id 1 --strategy balanced
```

### 9.3 三种策略

| 策略 | 闭关 | PK | 属性倾向 |
|------|------|-----|---------|
| aggressive | 1h 出关 | 主动打 | 攻击型 |
| defensive | 8h 长闭关 | 不打 PK | 防御型 |
| balanced | 4h 出关 | 适度打 | 均衡 |

### 9.4 Learning Tree（道果）

Agent 每次操作记录为 Merkle Tree 叶子，每 10 次操作自动上链存证——这就是你的 NFA 的链上行为证明。
