## 当前经济模型摘要

- 铸造 NFA：`0.005 BNB`
- NFA 总量：`2000`
- 初始最高：`金丹十二层 / Lv.36`
- 注灵收益：个人本金 `1%/天`
- PK 排名池：金库可释放余额 `0.2%/天`
- 灵气池：金库可释放余额 `0.1%/天`
- PK 手续费：总池 `3%`，全部销毁
- 邀请奖励：仅来自被邀请者注灵收益，比例 `3%`，手动领取
- PK 挂单退出：`10 分钟`
- 解除质押冷却：`24 小时`

## 长期方向

问道的长期目标不是单一链游，而是构建完整 Web3 修仙生态：

- 武器系统：法器、灵宝、古宝、通天灵宝
- 副本系统：秘境、洞府、遗迹、天劫幻境
- 宗门系统：宗门建设、宗门战、资源争夺
- NFA 交易系统：成长记录、战绩、灵根、Agent 行为一并沉淀
- 材料与锻造：副本掉落、强化、熔炼、丹药炼制
- AI Agent 网络：让 NFA 从游戏资产扩展成可代理行动的链上人格
