# 问道 · WenDao

> 凡人入链，大道可期。

**问道** 是一款基于 BAP-578 NFA 标准的修仙养成链游，部署在 BNB Smart Chain 上。

绑定你的 X 账号 → 天道审视你的 Web3 经历 → 决定你的初始境界与灵根 → 铸造链上修仙身份 → 闭关修炼、论道斗法、突破境界 → 天榜瓜分宝库。

**你在 Web3 的经历，就是你的修仙路。链会记住一切。**
